自动按需引入组件

March 19, 2022 · 2 min read

前言：使用unplugin-vue-components后，可以实现按需引入，而且也不用再使用import引入需要的组件，直接使用<el-button>等组件就可以。

支持的组件库：

  • Ant Design Vue
  • Arco Design Vue
  • Element Plus
  • Element UI
  • Headless UI
  • IDux
  • Inkline
  • Naive UI
  • Prime Vue
  • Vant
  • VEUI
  • Varlet UI
  • View UI
  • Vuetify
  • VueUse Components
  • Quasar
  • TDesign

1、安装npm包

npm install unplugin-vue-components unplugin-auto-import -D

2、配置

const AutoImport = require('unplugin-auto-import/webpack')
const Components = require('unplugin-vue-components/webpack')
const { ElementUiResolver } = require('unplugin-vue-components/resolvers')

module.exports = {
    configureWebpack: {
      plugins: [
        AutoImport({
          resolvers: [ElementUiResolver()],
        }),
        Components({
          resolvers: [ElementUiResolver()],
        }),
      ],
    }
}

3、具体使用

<template>
    <div>
        <el-button>按钮</el-button>
    </div>
</template>

4、确认是否按需引入

首先，安装webpack-bundle-analyzer插件，

npm install  webpack-bundle-analyzer -D

然后执行以下命令：

npm run build --report

不用做任何配置，就可以看到终端打印出来打包后各文件大小。

如果想看详细的打包后文件的信息，可以在vue.config.js中做如下配置：

const BundleAnalyzerPlugin = require('webpack-bundle-analyzer').BundleAnalyzerPlugin;

module.exports = {
    configureWebpack: {
        plugins: [
            new BundleAnalyzerPlugin()
        ]
    }
}

完整引入时，打包后chunk.js文件大小为400k左右。按需引入后，打包后chunk.js文件大小为200k左右。

5、非 template 中使用组件

如果在ts文件中使用ELMessage等组件，仍需要手动引入，代码如下。

import { ElMessage } from 'element-plus'
import 'element-plus/theme-chalk/el-message.css'
